# Chart Invaders

A retro arcade web game for Charta Health's promotional surface. Players shoot descending chart errors before they cross the pre-billing line.

## Scripts

- `npm run dev` starts the local Vite server.
- `npm run build` type-checks and creates a production build.
- `npm run test` runs game-logic tests.

The MVP is static, stores high score in `localStorage`, and uses no backend.
